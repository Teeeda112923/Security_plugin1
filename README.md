# Security_plugin1

WordPressのセキュリティ上、大切なポイントを分かりやすく伝え、利用者が少ない手間で対処できる環境を提供するためのプラグインです。

> 現在は企画・設計段階です。記載している機能は、今後の開発予定を含みます。

## 目指すもの

セキュリティは重要である一方、専門用語が多く、「何が問題なのか」「何をすればよいのか」が分かりにくい分野です。

本プラグインでは、単に危険性を知らせるのではなく、次の内容を読みやすい文章で伝えます。

- 現在どのような状態になっているか
- なぜ対策が必要なのか
- 放置すると何が起こる可能性があるか
- どのように対処すればよいか
- 対処するとサイトにどのような影響があるか

## 基本方針

- 難しい言葉を使わず、読みやすい文章で伝える
- 本当に大切な項目を優先して表示する
- できる限り少ない操作で対処できるようにする
- 設定変更による影響を事前に伝える
- 変更した設定を元に戻せるようにする
- サイトの使いやすさを損なわない
- Webサイトやサーバーへ過度な負荷をかけない
- 利用者の許可なく重要な機能を停止しない
- 過度に不安をあおらず、具体的な対処につなげる

## 想定する利用者

- 個人でWordPressブログを運営している人
- 小規模な企業や店舗のWebサイト担当者
- WordPressやセキュリティに詳しくない管理者
- 制作会社からWebサイトの管理を引き継いだ人
- 高額なセキュリティサービスを導入しにくい人
